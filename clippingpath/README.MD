```
Transition Follow This Process

1. 이미지의 초점 (즉, 주요 피사체)을 식별합니다.
2. 해당 개체에 대한 클리핑 패스 만들기
3. 경로로 다음 이미지 잘라 내기
4. 잘라낸 이미지 (실루엣)가 희미 해집니다.
5. 뷰포트보다 클 때까지 클리핑 경로의 크기를 조절하십시오.
6. 다음 이미지를 표시하려면 Trnasition을 완료하십시오.
7. Repeat!
```
